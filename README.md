## ✨ Latest Updates: Google-Inspired Refresh
We recently overhauled the visual experience to include:
- **Google-Style Confetti Animation**: A vibrant, dynamic particle background featuring Google's signature colors (Blue, Red, Yellow, Green).
- **Minimalist Branding**: Removed the legacy "DK" logo for a cleaner, modern look.

## 🛠 Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS
- **Icons**: Feather Icons
- **Animation**: Custom CSS Keyframes & JavaScript Particle Engine
- **Integrations**: Zoho CRM, Zoho Flow, Zoho Creator, REST APIs

## 📂 Project Structure
- `index.html`: Main landing page structure.
- `styles.css`: Custom styles, glassmorphism utilities, and animations.
- `script.js`: Interactive elements and the core confetti engine.
- `blogs.json`: Dynamic blog content data.
- `BuySell/`: [ASYNCRIX Global ERP Platform](file:///BuySell/README.md) - A full-featured Zoho-inspired cloud integration project.
